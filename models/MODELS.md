## How to Use These Models

1. Visit one of the resources above
2. Search for a model you like
3. Check if it's available in GLTF/GLB format
4. Download the model
5. Upload to your hosting
6. Create QR code pointing to the URL
7. Test with the AR app

## Export Format Requirements

For best compatibility:

- Format: GLTF 2.0 or GLB (binary GLTF)
- File Size: < 10MB (recommended < 5MB)
- Textures: Embedded in GLTF or hosted separately
- Lighting: Pre-baked or use PBR materials
