🧪 Testing Procedure

Test 1: Server Startup
  1. Run: npm run serve
  2. Verify: Server starts at http://localhost:8080
  3. Expected: Page loads without errors
Test 2: Demo Model Loading
  1. Click "Klik untuk Demo" button
  2. Verify: Colorful cube appears in 3D viewer
  3. Expected: Model loads in < 1 second
Test 3: Interactivity
  1. Model loaded (from Test 2)
  2. Drag model: Should rotate smoothly
  3. Scroll: Should zoom in/out
  4. Expected: Responsive, no lag
Test 4: Audio System
  1. Click "🔊 Audio" button
  2. Verify: Audio context initializes
  3. Expected: Beep sound plays, button shows "Playing"
Test 5: QR Code Generation
  1. Run: npm run generate-qr
  2. Check: QR images in public/qr/
  3. Expected: 3+ QR code PNG files generated
