QR membuka halaman AR, marker memunculkan objek 3D.

Gunakan proyek ini sebagai launcher experience AR. QR hanya berisi URL ke halaman ar.html?experience=..., lalu halaman AR membaca konfigurasi marker, model, audio, dan interaksi dari file JSON supaya mudah diperluas.

Coba demo Hiro Lihat daftar experience Unduh marker AR

Alur pemakaian

  1. Unduh marker di bagian bawah halaman ini, cetak atau tampilkan di layar lain.
  2. Scan QR dari perangkat mobile — browser membuka halaman AR.
  3. Izinkan kamera, tekan Mulai AR, lalu arahkan ke marker yang sudah disiapkan.
  4. Objek 3D muncul. Cubit untuk skala, geser untuk putar, ketuk 2x untuk reset.
💡 Cetak QR dan marker pada satu kartu/poster agar user cukup membuka QR sekali lalu langsung arahkan kamera ke marker di bawahnya.

Daftar Experience

Semua item di bawah dibaca dari public/data/experiences.json. Tambah entry baru untuk menambah QR, model, audio, dan marker.

Marker AR

Cetak atau tampilkan marker di bawah ini di layar lain. Arahkan kamera ke marker setelah AR aktif. Marker Hiro digunakan untuk experience demo-hiro, marker Kanji untuk demo-kanji.

Deploy ke GitHub Pages: Push ke branch main — GitHub Actions otomatis generate QR dan deploy. Jika mengganti nama repo atau username, update baseUrl di public/data/experiences.json lalu push ulang agar QR mengarah ke URL yang benar.
